## 🌟 Overview
Stop hunting for standard Persian web fonts. This repository contains ready-to-use CSS files and WOFF2 optimized assets for the most popular free Persian fonts (like Vazirmatn and Shabnam), pre-configured for modern web projects.

## ✨ Key Features
- 📦 **WOFF2 Optimized**: Maximum compression for lightning-fast web loads.
- 💅 **Ready-to-use CSS**: Drop-in `@font-face` declarations.
- 🆓 **100% Open Source**: Only includes commercially free, high-quality fonts.

## 🚀 Getting Started

### Prerequisites
Make sure you have `CSS_Fonts` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/awesome-persian-fonts.git
cd awesome-persian-fonts
No installation required. Just copy the CSS.
```

### Usage
```bash
<link rel="stylesheet" href="./style.css">
<style> body { font-family: 'Vazirmatn'; } </style>
```
