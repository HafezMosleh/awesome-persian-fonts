# awesome-persian-fonts

Curated list and ready-to-use CSS/WOFF2 files for the best free Persian fonts.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/awesome-persian-fonts.git
cd awesome-persian-fonts
```
